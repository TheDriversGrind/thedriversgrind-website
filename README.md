# TheDriversGrind Website

  Dark-mode landing page for [TheDriversGrind](https://thedriversgrind.com) — a gig driver community and Profit Tracker app ecosystem.

  ## Stack

  - React + Vite + TypeScript
  - Tailwind CSS v4
  - Framer Motion
  - shadcn/ui components
  - Wouter (routing)

  ## Goals

  - Convert gig drivers into Profit Tracker users
  - Grow YouTube subscribers
  - Grow Facebook community
  - Build email list via MailerLite
  - Drive Etsy shop traffic

  ## Dev

  ```bash
  pnpm install
  pnpm --filter @workspace/thedriversgrind run dev
  ```

  ## Config

  All external URLs are centralized in `src/config.ts`.
